Frequently Asked Queries

How do I export high-precision constants to LaTeX?

expand_more
Utilice la herramienta de exportación en la barra de navegación inferior o presione CTRL+E. Seleccione el formato 'LaTeX' y la precisión de decimales deseada para generar la cadena de código compatible con editores científicos.

¿Cuál es el límite de cómputo en la versión gratuita?

expand_more
La versión estándar permite hasta 10,000 operaciones por segundo con una precisión de 128 bits. Para simulaciones de Monte Carlo a gran escala o cálculos tensoriales complejos, considere el plan Workspace Pro.

¿Es posible integrar la API de CalcIng en entornos Python?

expand_more
Afirmativo. Contamos con un SDK oficial de Python disponible en PyPI. Ejecute pip install calcing-sdk para comenzar la integración con sus scripts de análisis de datos.
